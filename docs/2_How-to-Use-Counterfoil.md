# How to Use Counterfoil Kit in Your App

This guide will walk you through using Counterfoil Kit in a brand new React + TypeScript + Vite project. We'll assume you're starting from scratch and explain every step.

1. The [**Counterfoil Design Philosophy**](.1_Counterfoil-Design-Philosophy.md) has an overview of the guiding design principles for this kit: it is what we tried to keep in mind when putting the kit together. 
2. The [**How to Use Counterfoil**](2_How-to-Use-Counterfoil.md) document is this one! The central hub for our documentation.
3. The [**Set Up Counterfoil Kit**](3_Set-Up-Counterfoil-Kit.md) is a how-to guide for setting up the kit for the first time and updating it. 
4. The [**Component Overview**](4_Component-Overview.md) has all of the details (variants, props, etc...) for the components in the toolkit. 
5. The [**Troubleshooting Guide](5_Troubleshooting.md) has answers to common problems that happen when implementing Counterfoil. 


Happy coding! 🎉
